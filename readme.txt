On activation, you will see the ID of every post in the posts list.

== Description ==
When working with other plugins, shortcodes and widgets sometimes they ask for the post ID. And you would normally have to enter to the post or over on it to get it. This plugin adds a column in the post lists where you can see the post ID.

== Installation ==
1.- Download the file.
2.- Extract it.
3.- Upload it to wp-content/plugins/
4.- Go to Plugins in the Wordpress admin of your site.
5.- Click on \"Activate\".
